ABOUT WEBSITE

Brief Description - - Document Setup !DOCTYPE html, html, head These tags define the page as an HTML5 document, set language, character encoding, viewport for responsiveness, and link external CSS files. They prepare the environment before any visible content appears. - Page Content body Inside the body, you’ve used a combination of semantic tags nav, header, section and generic containers div, span to structure the menu. - The navigation bar nav, ul, li, a lets users move between pages. - The menu sections section, div, span, img, p, h1, h2 organize food categories and items. - Each food item is wrapped in a div with an image, name, and price, plus an onclick attribute to trigger JavaScript. - The bill area uses div, textarea, and button to show cart items, accept notes, and confirm orders. - The success screen is another div that appears after placing an order. - Interactivity script
